## Huskeregler for git repository
De viktigste filene å ha i git er:
 - configuration.yaml
 - customize.yaml
 - groups.yaml
 - scripts.yaml
 - automations.yaml

Den viktigste filen å ikke ha i git er:
 - secrets.yaml :-)

 Filer som blir dynamisk endret av systemet er ikke vits å ha i git. Se .gitignore for eksempel.
